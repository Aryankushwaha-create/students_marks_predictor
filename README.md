# 🎓 Student Marks Prediction

A Machine Learning web application that predicts a student's marks based on the number of study hours using **Linear Regression**.

## 🚀 Features

- Predict marks based on study hours
- Machine Learning model built using Linear Regression
- Interactive web application using Streamlit
- Easy-to-use interface

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## 📂 Project Structure

```
students_marks_prediction/
│── app.py
│── model.pkl
│── requirements.txt
│── README.md
│── data/
│── notebooks/
```

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python -m streamlit run app.py
   ```
